# Kashi Seva 🛕

**Smart Darshan Booking System** — built for Smart India Hackathon (SIH)

## Problem
Major temples in Varanasi face heavy, unmanaged crowding during darshan hours,
creating long wait times and safety risks for devotees.

## Solution
Kashi Seva lets devotees register for a darshan slot in advance using a simple
OTP-based verification flow — no app download, no complex signup. This spreads
footfall across time slots, reducing crowd density and easing the experience for
both devotees and temple administration.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend/Auth: Firebase (OTP verification, real-time data)

## How it works
1. Devotee enters phone number
2. OTP sent and verified via Firebase
3. Devotee selects an available darshan slot
4. Booking confirmed — reducing walk-in crowd load

## Status
Built as a hackathon prototype for SIH.
